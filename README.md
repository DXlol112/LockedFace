### О приложении
**LockedFace** — это настольное приложение на Python для анализа видеопотока с использованием технологий компьютерного зрения. Приложение обнаруживает лица и отслеживает состояние глаз пользователя в режиме реального времени, если пользователь отвернулся, то приложение воспроизводит выбранный файл.
<br> 
**Основная цель** — помочь пользователям не отвлекаться от работы, учёбы и других задач, требующих концентрации.

### Основные возможности
- **Распознавание лица** 
- **Отслеживание глаз (включается в настройках)** 
- **Поддержка очков в распознавании (включается в настройках)** 
- **Воспроизведение выбранного файла при отворачивании от экрана**
- **Возможность выбрать время работы приложения**
- **Простой интерфейс для настройки и управления приложением**
- **Пауза и продолжение работы приложения, без закрытия приложения**

### Используемые технологии:
| Компонент | Назначение |
| :--- | :--- |
| **Python 3.12** | Язык разработки |
| **OpenCV** | Обработка видеопотока |
| **MediaPipe** | Детекция лиц и ключевых точек |
| **NumPy** | Вычисления расстояний |
| **PyQt6** | Графический интерфейс (GUI) |
| **ffpyplayer** | Воспроизведение медиафайлов |

## ⚙️Установка<a id="installation"></a>
1. Скачайте из `Releases` последнюю версию приложения для вашей операционной системы.
2. Распакуйте архив и запустите **`LockedFace.exe`**
3. Следуйте инструкциям на экране.
4. В настройках выберите файл, и установите время работы приложения.
5. **Готово!**

>[!CAUTION]
>
>### Важно
>Иногда приложение может не сразу запустится, попробуйте перезапустить его, если возникнут проблемы с запуском.
><br>
><br>
> Приложение требует доступ к камере, для корректной работы. Убедитесь, что вы предоставили необходимые разрешения.

## 📂 Структура проекта
```text
├── log/                     # Логирование работы приложения
│   └── app.log
├── media/                   # Медиафайлы для тестов и демонстрации
├── script/                  # Исходный код приложения
│   ├── core/                # Основная логика и функции
│   │   └── def_collection.py
│   ├── style/               # Стили интерфейса (QSS)
│   │   ├── all_project.qss
│   │   └── dialog_win.qss
│   ├── ui/                  # Основные страницы интерфейса (PyQt/PySide)
│   │   ├── file_page.py
│   │   ├── main_page.py
│   │   ├── settings_page.py
│   │   └── start_page.py
│   └── support_ui/          # Вспомогательные виджеты и диалоговые окна
│       ├── file_card_widget.py
│       └── win_dialog.py
├── static/                  # Статические графические ресурсы
│   ├── btn_icon/            # Иконки для кнопок управления
│   └── icon/                # Главные логотипы приложения (ICO, SVG)
├── config.json              # Конфигурационный файл приложения
├── lockedface.py            # Главный скрипт запуска (Точка входа)
├── requirements.txt         # Зависимости проекта
└── README.md                # Документация проекта
```
Некоторые папки появляются после первого запуска приложения, например папка `log` для логирования.
